# COVID-19 Dashboard Analysis

## About This Project

I created this COVID-19 Dashboard using Tableau and Microsoft Excel to analyze and visualize global COVID-19 data. The dashboard helps understand COVID-19 cases, deaths, daily trends, and the top affected countries through interactive charts and filters.

## Tools Used
- Tableau
- Microsoft Excel

## Features
- Global COVID-19 Cases Trend
- Global Deaths Trend
- Daily New Cases Trend
- Top 10 Countries by Total Cases
- Interactive Location Filter
- Dashboard Insights

## Files Included
- Covid_TABLEAU.twbx
- Dataset.xlsx
- Dashboard.xlsx
- Dashboard_1.jpg

## What I Learned
- Creating interactive dashboards in Tableau
- Data visualization techniques
- Trend analysis
- Working with filters and charts
- Presenting data in a clear and meaningful way
